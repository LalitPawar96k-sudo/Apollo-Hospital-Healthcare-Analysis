# Apollo-Hospital-Healthcare-Analysis
# 🏥 Apollo Hospital Healthcare Analysis Dashboard | Power BI

## 📌 Project Overview

The **Apollo Hospital Healthcare Analysis Dashboard** is an interactive Power BI project developed to analyze hospital operations, patient care, admissions, billing, and overall healthcare performance. The dashboard provides actionable insights that help hospital administrators monitor key performance indicators (KPIs), improve operational efficiency, and support data-driven decision-making.

This project demonstrates the application of **Business Intelligence (BI)** techniques in the healthcare domain using Power BI.

---

## 🎯 Project Objectives

- Analyze patient admission trends.
- Monitor hospital revenue and billing performance.
- Evaluate patient demographics.
- Track doctor and department performance.
- Identify treatment and diagnosis patterns.
- Improve healthcare decision-making through interactive dashboards.

## 📊 Dashboard Insights

This dashboard helps answer important healthcare questions, including:

- How many patients were admitted over a selected period?
- Which departments receive the highest number of patients?
- What is the total hospital revenue?
- Which doctors handle the most patients?
- What is the average treatment cost?
- How efficiently are hospital beds utilized?
- Which patient age groups visit the hospital most frequently?
- What are the monthly admission and discharge trends?
